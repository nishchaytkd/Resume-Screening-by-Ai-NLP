# Resume Screening AI

This project classifies resumes into job categories using NLP and Machine Learning.

Features:
- Upload PDF resume
- Extract resume text
- Predict job category
- Built with Streamlit

Technologies:
- Python
- Scikit-learn
- TF-IDF
- Streamlit

## Dataset

Download the dataset here:  
[Download Dataset](https://drive.google.com/file/d/11B3GcEWWAMILMCoGMCN3zfSxm57a5ViN/view?usp=drive_link)

## How to Run the Project

1. Install dependencies:

pip install -r requirements.txt

2. Train the model:

python NLP.py

3. Run the Streamlit app:

streamlit run app.py
